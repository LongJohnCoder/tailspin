Javascript in Javascript

Tailspin is a Javascript interpreter written in Javascript. It runs in the browser and sandboxes the running code so that it doesn't pollute the host web page. Why? Because it allows for some really interesting interactive programming experiences.

  • Interactive Javascript Problems
  • Algorithm Visualisation Wiki

Pause-able

Tailspin is written in continuation passing style which allows the interpretation of code to be paused and controlled. It also means that running code in Tailspin does not block the browser's Javascript execution, because Tailspin pauses regularly to allow the browser to run other code.

Reversible Javascript

Tailspin is able to run in a reversible mode, where state changes are recorded so that they can be undone. Using Tailspin's reversibility code can be stepped through both forward and backward.

Standards compliant

Tailspin currently passes 99.9% of the ECMAScript Language test262 test suite.
